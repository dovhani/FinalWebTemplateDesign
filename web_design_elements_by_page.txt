================================================================================
  PART 10 — GLOBAL UI PATTERNS (Reused Across Pages)
================================================================================

  The following UI building blocks appear on multiple pages:

  GP-1.  Section Heading / Title Block
         Used on every content section.
         Built from: eyebrow label + H2 + decorative rule line.

  GP-2.  Page Hero Header
         Full-width dark banner with background image.
         Built from: background image div + eyebrow badge + H1 + tagline.

  GP-3.  Info / Feature Card
         Generic boxed container (shadow, border, rounded corners).
         Variants: value card, facility card, department card, contact card.

  GP-4.  Profile Card (Staff / SGB / Security / Ground / Food)
         Overlapping avatar + role badge + name + CTA button.
         Opens a Detail Modal on click.

  GP-5.  Detail Modal Overlay
         Fixed, full-screen, backdrop-blurred overlay.
         Built from: backdrop + content panel + image panel + info panel.
         Closes via: backdrop click OR close button OR Escape key.

  GP-6.  Scroll Reveal Animation (.ws-reveal / .sh-reveal)
         Applied via: IntersectionObserver in index.astro + ScrollReveal.astro
         Effect: opacity 0 → 1 + translateY 20px → 0, staggered by index.

  GP-7.  Stats Bar
         Dark glassmorphism grid displaying key school numbers.
         Used on: Hero (home), Our School page.

  GP-8.  CTA Button
         Distinct styled button (maroon/gold) driving user action.
         Variants: "Enrol Now", "Contact Us", "View Gallery", "Profile".

  GP-9.  Accordion Item (FAQ)
         Toggle button + collapsible answer panel.
         Used on: Enrolment page.

  GP-10. Form (Contact / Enrolment)
         Labelled input fields + select dropdowns + textarea + submit button.
         Includes client-side validation and submit handler.
